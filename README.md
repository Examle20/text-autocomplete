# LSTM Text Autocomplete

---

## Цель проекта

Создать нейросеть, которая на основе начала фразы предсказывает её продолжение.

---

## Выполненные задачи

1. Взят датасет с короткими постами (https://code.s3.yandex.net/deep-learning/tweets.txt). На его основе были подготовлены данные для обучения
2. Реализована модель на основе LSTM
3. Модель была обучена на подготовленных данных, были сняты метрики: точность, ROUGE1 и ROUGE2.
4. Была проверена генерация текста на случайных промптах
5. Была взята модель distilgpt2 из Transformers. Для нее были измерены метрики ROUGE1 и ROUGE2
6. Сделаны выводы о работе двух моделей
